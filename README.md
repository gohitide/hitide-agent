# HiTide Agent Skill

Teach an AI agent (Claude Code, etc.) to drive your [HiTide](https://gohitide.com) workspace — campaigns, subscribers, metrics, and integrations — all as JSON.

HiTide is **social SMS list growth**: turn your Instagram, TikTok & Facebook followers into SMS & email subscribers with automated DMs.

## Install

```bash
npx skills add gohitide/hitide-agent
```

The skill bootstraps the CLI on first use (`npm install -g @hitide/cli`), then authenticates with your Workspace API key.

## Get your API key

At [gohitide.com](https://gohitide.com), log in → **Settings → API Keys → Manage API Keys → Create API Key** (it starts with `ws_`).
